Tuple - Krotka

nawias: okrągły ( ), brak nawiasów, choć według PEP 8 zaleca się stosowanie okrągłych

cecha istotna: (immutable)! - niezmienna

Krotki, podobnie jak listy, mogą zawierać mieszane typy danych (np. liczby całkowite, zmiennoprzecinkowe, tekst lub a listy).



tuple = (1,2,4,4,5,6)
tuple2 = 1,4,5
days = tuple(["pon","wto","śro","czw","pią","sob","nie"])

# Krotka stworzona z różnych typów
tup_mixed = (1, 'dwa', [3, 4.1])

Cechy:

Podobna do listy, z tą różnicą, że gdy raz się zadeklaruje element, nie można go już modyfikować (czyli jest niemutowalny). Krotka służy więc do przechowywania różnego rodzaju elementów, których nie będziemy zamieniać lub przestawiać.

Struktura niezmienna, wykorzystywana do przechowywania danych niezmiennych w czasie jak np. współrzędne gps.


# 'Zbiorowe' definiowanie zmiennych
belka1 = ('C30/37', 0.3, 0.5)	# parametry belki: klasa betonu, szerokość, wysokość
(klasa_bet, b, h) = belka1

#Teraz można bezpośrednio odwoływać się do poszczególnych parametrów belki
print(klasa_bet)
print(b)
print(h)

Krotki, podobnie jak listy, mogą zawierać mieszane typy danych (np. liczby całkowite, zmiennoprzecinkowe, tekst lub a listy).

Tuple opierają się na indexsach co pozwala im zachowywać zduplikowane elementy

Krotki możemy używać w słownikach jako ich klucze.

Krotka jest niezmienna, nie wywołamy na niej metod: append, remove(), extend() czy pop()

Indeksowanie

Tuple opierają się na indexsach co pozwala im zachowywać zduplikowane elementy


print(tup[0])    # odwołanie do pierwszego elementu krotki
print(tup[-1])   # odwołanie do ostatniego elementu
print(tup[1:])   # wyodrębnienie elementów krotki od drugiego do ostatniego

days = ("pon", "wto", "śro")
day_1, day_2, day_3 = days
print(day_2)

Iterowanie po krotce

po tuplach możemy iterowć, podobnie jak po listach, przeglądając elementy kolekcji



tuple = (1, 6, 5, 3, 8)
for i in tuple
    print(i)

Operacje na krotkach 

tup = ('jeden', 'dwa', 'trzy')
print(tup)

# Zwraca liczbę elementów krotki
print(len(tup))

# Zwraca indeks pierwszego wystąpienia szukanego elementu w krotce
print(tup.index('dwa'))

# Zwraca liczbę wystąpień elementu w krotce
print(tup.count('dwa'))

# Prawda (True) jeśli element występuje, w przeciwnym przypadku fałsz (False)
print('element' in tup)

# Zwraca najmniejszy min() i największy max() element krotki
tup_numbers = (1,2,3)   # żeby użyć tych metod krotka musi zawierać tylko elementy liczbowe
print(tup_numbers)
print(min(tup_numbers))
print(max(tup_numbers))

# Zwraca sumę elementów krotki
print(sum(tup_numbers))

print(tup1 + tup2)

Sprawdzanie, czy jakiś element znajduje się na liście krotki za pomocą in:




krotka = 34, 45.6,"tekst"

if"tekst" in krotka:
    print("Znaleziono w krotce ten element!")
else:
    print("Nie znaleziono w krotce tego elementu")

Znajdowanie indeksu pierwszego elementu w krotce za pomocą metody wewnętrznej index:



krotka = (34, 45.6,"tekst")

if "tekst" in krotka:
    print("Znaleziono w krotce ten element pod indeksem {indeks}!".format(indeks = krotka.index("tekst")))

Zliczanie wystąpień w krotce danej wartości za pomocą metody wewnętrznej count:



krotka = (2, 4, 6, 7, 7, 7)
print("Znaleziono w krotce {indeks} elementów zawierających liczbę 7!".format(indeks = krotka.count(7)))

Uzyskiwanie krotki odwróconej:



krotka = (2, 4, 6, 7, 7, 7)
print(krotka[::-1])

Slicing - Uzyskiwanie wycinka krotki:



krotka = (2, 4, 6, 7, 7, 7)
print(krotka[1:4])
